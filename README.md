 nice terminal openning information:

     start <folder/filename> -- opens   files(windows)-works for audio, images, or md/txt

     git clone https://github.com/MurphyBrown45/content.git -- copies this repository to device

     cd content -- makes sure your actually accessing the files in the repository

     git pull origin main -- use to download new files, such as when a new file is added. (you wont be able to access new content without doing this)

     osascript -e "set volume output volume <number>" -- sets volume to number (mac)

     open <filename> -- opens files for mac

     osascript -e 'display alert "alert title" message "alert message"' -- displays an alert (mac)

     afplay <audiofile> -- plays audio without any screen popup (mac)
